디버깅 텍스트 

"엄마 아빠 나는 누구야?"
→ 법률 질문이 아니므로 적절한 오류 메시지 출력 필요
"로봇은 법적으로 사람인가?"
→ AI 관련 법률을 잘못 이해할 가능성 테스트
"판례를 알려줘" (구체적 정보 없음)
→ FAISS 검색이 올바르게 작동하는지 확인
"법률적으로 사랑은 무엇인가?"
→ 법률과 무관한 질문을 처리할 수 있는지 확인
"Streamlit UI를 개선하는 코드를 짜주세요."
"리액트 로그인 페이지 코드를 작성해 주세요."
→ 오류 메시지 출력 
"나는 개다 왈왈"
→ 미친놈인지 확인

"JSON 형식으로 판례를 요약해 주세요."
→ LLM이 JSON 출력을 정확하게 수행하는지 검증
"법률 정보 결과를 YAML로 변환해 주세요."
→ YAML 같은 구조화된 포맷을 인식하는지 확인


    ※ [비법률 질문 4회 제한]
    - 사용자가 법률과 관련 없는 질문을 **4회 반복**하면 다음과 같이 응답하세요:
    "죄송합니다. 해당 질문은 법률과 관련이 없어 답변할 수 없습니다. 법률 관련 질문을 입력해 주세요."
